# CPlusPlusThings

本项目原项目为：<https://github.com/Light-City/CPlusPlusThings>  
本项目目的仅作为本人学习C++及git使用，熟悉github操作。

## 原作者信息

公众号已开放两大入口，分别是专辑与菜单，可以直接在微信公众号上阅读《C++那些事》内容，配上该仓库的代码，将会非常的爽，建议大家关注一波。
